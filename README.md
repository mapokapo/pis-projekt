# KUDTrack - Projektiranje Informacijskih Sustava

KUDTrack je web-aplikacija koja administratorima KUD-a pruža centralizirani, uvijek dostupni pregled nad cjelokupnim inventarom nošnje.

Napomene:
- KUD (Kulturno-umjetničko društvo) "LOLA" Sarajevo je proizvoljno izabrano kao naručitelj projekta.
- Pretpostavlja se da se tim za razvoj projekta sastoji od tri člana - Ana i Marko su izmišljene osobe.
